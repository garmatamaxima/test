
### *PowerToys Диспетчер Клавіатури*

CapsLock -> win + space // Змінює мову 
Left Alt + Space -> win + PageUp // Відсилає команду до **PowerToys Fancy zones** щоб змінити зображену програму в зоні.

### *PowerToys Fancy zones*

ctrl + win + alt + 1-9 та 0 // пресети зон

---
### *Профі кейбінди*

win + tab // Перегляд робочих просторів
ctrl + win + <- або ctrl + win + -> // перехід між робочими просторами
win + alt + space // **Windows command pallete** команда
